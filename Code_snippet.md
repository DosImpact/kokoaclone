- 박스 쉐도우

box-shadow: x오프셋 | y오프셋 | 그라데이션 상수 | 그림자 크기 | {inset 안으로 그림자}
참고로 여러개의 쉐도우를 적용하고 싶으면, 하나의 프로퍼티안에 여러개를,로 연결시켜 주어야한다.
box-shodow를 또 쓰게 되면 아예 오버로딩 되버림;;

```
  box-shadow: -100px -500px 2000px white inset;

```
